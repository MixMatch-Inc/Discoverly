# Discoverly Mobile

Active mobile stack for contributors:

- Expo
- React Native
- TypeScript
- Expo Router
- Zustand

## Quick Start

1. Copy `.env.example` to `.env`
2. Install dependencies:

```bash
npm install
```

3. Start app:

```bash
npm run start
```

## Notes

- Deep link scheme is `discoverly`.
- EAS profiles are defined in `eas.json`.
